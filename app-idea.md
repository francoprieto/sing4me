# Sing4Me App

Is a mobil application that allows you to save song lyrics locally in your device. It allows you to display the lyric you want, at a live show or in rehearsal, avoiding the screen to turn itself off (sleep) approximately for the duration of the song.

## Tools:

The application must be developed using react-native with support for iOS and Android devices. It has to take advantage of the device entire screen.

## Features:

1. The app's main screen should display a list of songs (by their titles) that were added or created by the user. Also it must have an option to add a new song lyric. 
Each created song in the list must have the option to edit, an option to configure and an option to view the song lyrics.

2. When the user clicks on the "Add new song lyric" option, the app must allow the user to add a title, and the lyrics for the new song. The user can paste the lyrics from the clipboard or write them from scratch. The user can re-arrange the lyrics as the user wants.

3. When the user clicks on the "Edit" option of a particular song, the app must allow the user to change the lyrics and save the changes.

4. When the user clicks on the "Configure" option of a particular song the app must allow the user to:

- highlight part of the lyric with different colors
- increase or decrease the font size of that particular lyric
- Change the background color of that particular song

5. When the user clicks on the "View" option, the lyrics must display according to its configuration (view item 4). When the app is in "View" mode, it must avoid the screen to sleep (not to turn off).
