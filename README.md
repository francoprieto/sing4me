# Sing4Me

A React Native app built with Expo to save song lyrics locally, configure appearance, and view lyrics without the screen sleeping.

## Setup

1. Install dependencies:
   ``npm install``
2. Run locally:
   ``npm start``

## Features

- Add, edit, configure, and view saved song lyrics
- Rearrange lyrics lines manually
- Highlight lyric lines with selectable colors
- Adjust font size and background color per song
- Keep the screen awake while viewing lyrics
